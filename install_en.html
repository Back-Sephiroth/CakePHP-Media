Plugin Media

For CakePHP
Introduction Installation Usage French

The first thing you have to do is download the plugin and put it in your Plugin directory, you can clone it using git if you have git installed

git clone git@github.com:Grafikart/CakePHP-Media.git Media

After that you have to load the plugin, in your bootstrap.php file you have to uncomment

CakePlugin::loadAll();

// Or
CakePlugin::load('Media');

Done ! The plugin is now loaded in your application, you know have to import the table that will save the medias. To do that you have to use CakePHP CLI : 

cake Media.create

And that's all you are ready to go !
