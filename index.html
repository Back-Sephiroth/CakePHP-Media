Plugin Media

Pour CakePHP
Présentation Installation Utilisation Configuration

Le principe de ce plugin CakePHP est de vous permettre de créer une gestion d'image poussée pour vos contenu avec le moins de code possible. Le système repose sur un behaviour pour gérer l'association des média aux models et un helper pour gérer la partie administration des médias.

Ce plugin est essentiellement conçu pour une utilisation en backend (toute la partie upload utilise le prefix "admin")

Fonctionnalités

  • Association belongsTo et hasMany Media pour vos model
  • Interface d'upload drag & drop des images pour le backoffice, basé sur plupload
  • Intégration de l'uploader d'image dans tinyMCE
  • Redimensionnement d'images à la volée
